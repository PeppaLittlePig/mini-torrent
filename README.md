# mini-torrent

开发环境：py3

相关依赖：requests,lxml,math,re

==========================================

运行: python command.py -k 电影名 -n 10 -o result.txt -s 1

-k keyword 电影名

-n num 需要获取的数据条数 <=200条

-o output 保存数据的文件名 暂支持文本文件 *.txt

-s sort 排序 默认0 按照日期排序，1 按照电影大小 2 按照电影热度
